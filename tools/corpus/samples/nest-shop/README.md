# nest-shop

Petit service e-commerce d'exemple, style **NestJS**, pour le corpus de démonstration de
CodeWorld Explorer. Architecture en couches : contrôleurs HTTP, services métier, accès aux
données, domaine et modèles. Certains dossiers portent un nom conventionnel (classés par
règle), d'autres un nom neutre classé d'après le contenu de leurs fichiers (heuristiques
statiques, couche 3).

Code fictif, sans dépendance installée : sert uniquement d'arborescence à analyser.
