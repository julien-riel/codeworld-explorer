# react-dashboard

Petit tableau de bord d'exemple, style **React**, pour le corpus de démonstration de
CodeWorld Explorer. Composants de visualisation, hooks, services d'accès à l'API et
utilitaires. Les dossiers `services/` et `lib/` sont classés par règle de nom ; `panels/`
et `hooks/` sont classés d'après leur contenu (imports React, composants) par les
heuristiques statiques (couche 3).

Code fictif, sans dépendance installée : sert uniquement d'arborescence à analyser.
